# CRITICAL: JAX must stay <0.11. PennyLane 0.45 imports jax.core symbols

# This pairing (PennyLane 0.45.1 + JAX/jaxlib 0.10.0) is verified working on CPU.
# For GPU, install `jax[cuda12]==0.10.0` matched to the node's CUDA instead (and
# beware clashes with the CUDA torch build above); CPU jax is fine for the
# <=18-qubit statevector sims these models use.
